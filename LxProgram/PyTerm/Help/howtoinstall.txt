PyTerm Yazılımını Nasıl Kurarım?: (Not: Program Sadece Linux İçindir)

Adımlar:

1. dosya dizininde Terminali açınız Ve ardından chmod a+x installpytrm.sh komutunu kullanınız.
2. ./installpytrm.sh yazıp dosyayı çalıştırınız. Bu dosya "PyTerm" Yazılımını Çalıştırabilmeniz için pytrm adlı yazılımı yükler.

Çalıştırmak İçin:

1. Terminali açınız.
2. pytrm yazınız ve işletim sistemini Giriniz.
3. Ardından Yazılım Çalıştırılcaktır. (Not: Şuan Yazılım Önizleme Aşamasındadır Ve Sadece TR Sürümüdür.)



How Do I Install PyTerm Software?: (Note: The Program Is Only For Linux)

Steps:

1. Open Terminal in the file directory and then use the chmod a+x installpytrm.sh command.
2. Type ./installpytrm.sh and run the file. This file installs the software called pytrm so that you can run the "PyTerm" Software.

To make it work:

1. Open the terminal.
2. Type pytrm and enter the operating system.
3. Then the software will be run. (Note: Currently the software is in the preview phase and it is only the TR version.)
