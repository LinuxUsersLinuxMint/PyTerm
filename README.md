# PyTerm

* PyTerm (Sadece Linux İçindir ancak kaynak kodundan derlenip diğer işletim sistemleri içinde kullanılabilir) Yazılımları/Uygulamaları Başka klasörlerden çalıştırmayı sağlar.
Kullanım Örneği (PyTerm):

PyTerm yazılımını açınız ve Bir /path/ ve path_var ekleyiniz. Eklemek İçin "FileAdd" Kullanabilirsiniz.
 Örnek:
  * İşlem Giriniz: FileAdd
  * Yol giriniz: "/LxProgram/PyTerm/PyTerm" (Tırnak işaretleri olmadan) / (Dosya dizini sadece bir örnektir.)
  * Değişken giriniz: "path_py_term"  (Tırnak İşaretleri olmadan)

* Yazılımın Doğru çalışması için başlangıç uygulamalarına eklemeniz ve yazılımı arka planda çalıştırmanız gerekir. (Yazılım file_path verilerini kaydetmez (Bunun üzerinde çalışma yapacağız))
* Yazılım Sizden Herhangi bir veri talep etmez ve sizden Kullanıcı Verileri almaz. (Yazılım tamamen açık kaynaklıdır isterseniz bunu kendiniz derleyebilirsiniz)
* Yazılım Şu anda "Stable BETA" Sürümündedir.

* PyTerm (Only for Linux, but can be compiled from source code and used in other operating systems) Allows to run Software/Applications from other folders.
Usage Example (PyTerm):

Open PyTerm and add a /path/ and path_var. You can use "FileAdd" to add.
 Example:
  * Enter Action: FileAdd
  * Enter path: "/LxProgram/PyTerm/PyTerm" (Without quotes) / (The file directory is just an example.)
  * Enter variable: "path_py_term" (without quotes)

* For the software to work correctly, you must add it to the startup applications and run the software in the background. (The software does not save file_path data (We will work on this))
* The Software does not request any data from You and does not receive User Data from You. (The software is completely open source, you can compile it yourself if you want)
* The Software is Currently in "Stable BETA" Version.
